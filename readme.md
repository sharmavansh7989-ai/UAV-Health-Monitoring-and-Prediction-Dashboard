An intelligent AI-powered system designed to monitor, analyze, and predict the health status of Unmanned Aerial Vehicles (UAVs) in real time. This project combines machine learning, data analytics, and interactive visualization to enhance reliability and safety in UAV operations.

---

## Project Overview

The **UAV Health Monitoring and Prediction Dashboard** is built to simulate an industrial-grade monitoring system that:

* Tracks UAV system parameters
* Detects anomalies in real time
* Predicts system health status
* Provides actionable diagnostics

This system is particularly useful for:

* Aerospace monitoring systems
* Industrial drone operations
* Predictive maintenance solutions

---

## Key Features

* **Real-Time Monitoring Dashboard**
  Interactive UI for visualizing UAV parameters

* **AI-Based Prediction System**
  Machine learning model predicts system health

* **Anomaly Detection**
  Identifies abnormal patterns in UAV behavior

* **Health Score Calculation**
  Provides overall system reliability score

* **Diagnostic Insights**
  Explains possible causes of system degradation

* **Data Visualization**
  Graphs and trends using matplotlib

## Tech Stack

* **Frontend/UI**: Streamlit
* **Backend**: Python
* **Libraries**:
  * Pandas
  * NumPy
  * Scikit-learn
  * Matplotlib
  
* **Model Storage**: Joblib

## Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/sharmavansh7989-ai/UAV-Health-Monitoring-and-Prediction-Dashboard.git
cd UAV-Health-Monitoring-and-Prediction-Dashboard
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the application

```
streamlit run app.py
```
